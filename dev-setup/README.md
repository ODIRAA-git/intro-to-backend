# Fitness Moves — Frontend (React + TypeScript)

This repository contains a lightweight React + TypeScript frontend scaffold for a fitness app that lists workouts and demonstrates the moves in each workout.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

The app will default to Vite port 5173. Next steps: add a Node.js + MongoDB backend and REST/GraphQL APIs for workouts and moves.
